# 🎨 Maria's Artes | Atelier Criativo

Uma Landing Page de alta conversão desenvolvida para um atelier de artesanato. O projeto foi pensado como uma vitrine digital, com foco em apresentar os produtos de forma visualmente atrativa e facilitar o contato com clientes.

## 🧵 Produtos / Serviços

• Artesanato personalizado  
• Peças decorativas  
• Encomendas sob medida  
• Produtos exclusivos feitos à mão    
• Atendimento direto via WhatsApp  

---

## 🎯 Foco do Projeto e Características

Diferente de sistemas tradicionais, esta aplicação foi desenvolvida como uma landing page voltada para conversão e apresentação visual. Os principais destaques incluem:

- 🚀 **Foco em Conversão:** Botões estratégicos e integração com WhatsApp para contato rápido.
- 📱 **Mobile-First & Responsividade:** Interface adaptada para celulares, tablets e desktops.
- 🎨 **Design Atrativo:** Layout visual pensado para destacar os produtos artesanais e valorizar o trabalho manual.
- ⚡ **Performance Otimizada:** Carregamento rápido e experiência fluida para o usuário.
- 🧩 **Estrutura SPA:** Navegação fluida com fallback via `.htaccess` para ambientes Apache.
